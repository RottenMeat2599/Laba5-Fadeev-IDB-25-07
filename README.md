# Laba5-Fadeev-IDB-25-07

**Вывод**

В ходе выполнения лабораторной работы были реализованы все основные виды обхода бинарного дерева: прямой (preorder), симметричный (inorder), обратный (postorder) и обход в ширину (level order). Также были реализованы функции для нахождения максимального и минимального значений в дереве.

В результате тестирования на двух различных бинарных деревьях все функции отработали корректно. Для каждого дерева были получены соответствующие последовательности обхода узлов, а также верно определены максимальный и минимальный элементы. Программа успешно справляется с поставленной задачей.
